Propriedades CSS

Propriedades de Estilo de Texto

  • color — Define a cor do texto.
  • font-size — Tamanho da fonte (px, em, rem etc.).
  • font-family — Tipo de fonte utilizada.
  • font-weight — Espessura da fonte (normal, bold, 400…).
  • text-align — Alinhamento do texto (left, center, right, justify).
  • text-decoration — Adiciona linhas ao texto (underline, line-through).
  • line-height — Altura da linha, controla espaçamento vertical.

Propriedade de Caixa (Box Model)

  • width / height — Largura e altura do elemento.
  • margin — Espaçamento externo do elemento.
  • padding — Espaçamento interno.
  • border — Borda em volta do elemento (largura, estilo, cor).
  • box-sizing — Controla como width/height são calculados (content-box, border-box).
